===================
酒ソムリエ (試作版)
===================


----
起動方法

SakeSommelier.exe [/P ポート番号] /K GeminiAPIキー

	ポート番号    ... HTTPサーバーとして待ち受けするポート番号
	GeminiAPIキー ... Gemini API Key


----
終了方法

SakeSommelier.exe /S


----
実行例

SakeSommelier.exe /P 80 /K AIza***********************************
                           ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
                           ご自身の Gemini API Key を指定

⇒　http://localhost にアクセス
